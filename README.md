# HTC Raider/Velocity/Vivid patches for LineageOS 11.0
You must apply all of the patches in this repository in order for proper building and booting. 

Only validated to work on LineageOS 11.0 (as of Apr 2026)

Extract files to ~/holiday_patches. Copy apply.sh to the root of your LineageOS 11.0 sources.

Mark apply.sh as executable

```
sudo chmod +x apply.sh
```

Then run the apply.sh script. It should apply all of the patches you need

DISCLAIMER: I applied all of Team Hydra's patches. Some of these might not be needed. 

Thanks to @fakemanoan for the script and README, and thanks to Team-Hydra for the patches.
